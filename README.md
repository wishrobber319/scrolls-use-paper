# RimWorld of Magic Scrolls Use Paper

Moves A RimWorld of Magic's scroll, book and skill crafting off its own magic printer and onto Medieval Overhaul's book scribe table. Each recipe now costs paper instead of cloth/devilstrand, the old magic printer is removed, and the Scribing research now requires the Presser.

---

A RimWorld 1.6 mod by wishRobber.
